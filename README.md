# プロジェクト概要

正規表現で指定したURLの表示をブロックするChrome拡張機能

## 機能要件

1. URL ブロック機能
   - 正規表現パターンマッチング
   - ユーザーが指定した正規表現パターンと一致するURLを検出
   - パターンは複数登録可能（リスト形式で管理）
2. ブロック時の代替表示
   - ブロックされたページの代わりに静的なページを表示する
3. 時間制限機能
   - URLごとに1日の利用可能時間を設定
   - 分単位で設定可能（例：30分/日）
   - アクティブタブの滞在時間を計測（バックグラウンドタブは除外）

## 画面要件

### オプション画面

- ブロックルール入力セクション
  - ブロック対象URLの正規表現の入力フィールド
  - 利用可能時間（分単位）の入力フィールド
  - 追加ボタン

- ブロックパターンリストセクション
   - ブロックルールのリスト
     - ブロックルールアイテム
       - ブロック対象URLの正規表現の入力フィールド
       - 利用可能時間（分単位）の入力フィールド
       - 更新ボタン
       - 削除ボタン

### ポップアップ画面
なし

### ブロック画面
- 「ブロックされました」メッセージ
- 該当するブロックルールの表示
